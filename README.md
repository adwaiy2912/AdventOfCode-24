
# Advent of Code 2024 - Java Solutions

This repository contains my Java solutions for Advent of Code 2024.

## Structure

- `src/` - Java source files (`Day1.java`, `Day2.java`, ..., `InputManager.java`)
- `bin/` - Compiled `.class` files
- `input/` - Input files for each day
- `lib/` - (Empty for now, reserved for any libraries if needed)
- `README.md` - This file

## How to Run

1. Compile:
   ```bash
   javac -d bin src/*.java
   ```

2. Run a specific day:
   ```bash
   java -cp bin Day1
   ```

Inputs are managed through the `InputManager` class.
